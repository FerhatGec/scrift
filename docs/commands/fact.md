# Factorial Command

fact command is add text in file.

# Examples

```
input:
    <username>@<hostname>:~/home/<username> $# fact 5
output:
    120
    <username>@<hostname>:~/home/<username> $#
```

```
input:
    <username>@<hostname>:~/home/<username> $# fact -5
output:
    n must be > or = to 0
    <username>@<hostname>:~/home/<username> $#
```
