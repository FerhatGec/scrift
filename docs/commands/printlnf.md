# Printlnf Command

printlnf command is prints your input.

# Examples

```
input:
     <username>@<hostname>:~/home/<username> $# printlnf Hello Scrift!

output:
    Hello Scrift!
    <username>@<hostname>:~/home/<username> $#
```

# Example for Keywords

```
input:
      <username>@<hostname>:~/home/<username> $# printlnf #USER
output:
    <username>
    <username>@<hostname>:~/home/<username> $#
```
