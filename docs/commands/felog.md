# Fegeya Log Command 

* felog

* This commands is shows your logs (not logs, this is FeLogs :^))

# Example:
```
input:
     <username>@<hostname>:~/home/<username> $# felog

output:
    FeLog Started. 2020-40-01.21:40:16
    FeLog Started. 2020-41-01.21:41:55
    FeLog Started. 2020-47-01.21:47:28
```
