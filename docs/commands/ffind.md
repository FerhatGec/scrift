# Ffind Command

ffind command is finds files and directories.

# Examples 

```
input:
    <username>@<hostname>:~/home/<username> $# ffind .scrift
output:
    Settings*: .scrift_settings
    FeLog*: .scrift_log
    History*: .scrift_history
    Ascii Art*: .scrift_ascii
    <username>@<hostname>:~/home/<username> $#
```
