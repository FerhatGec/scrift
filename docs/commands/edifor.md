# Edifor Command

edifor command is simple Scrift-Based Text Editor

# Examples

```
input:
    <username>@<hostname>:~/home/<username> $# edifor
output:
    <Edifor>
    <username>@<hostname>:~/home/<username> $#
```

# Examples 2

```
input:
    <username>@<hostname>:~/home/<username> $# edifor <file>
output:
    <Edifor and output <file>>
    <username>@<hostname>:~/home/<username> $#
```
