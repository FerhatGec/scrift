# Ffind Command

ffind command is finds files and directories.

# Examples 

```
input:
    <username>@<hostname>:~/home/<username> $# ffind .scrift
output:
    [File]: .scrift_settings
    [File]: .scrift_log
    [File]: .scrift_history
    [File]: .scrift_welcome
    [File]: .scrift_ascii
    <username>@<hostname>:~/home/<username> $#
```
